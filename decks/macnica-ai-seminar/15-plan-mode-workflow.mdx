# Plan Mode ワークフロー
<p style={{ fontSize: "2rem", color: "var(--slide-text-muted)", marginTop: "-0.2rem" }}>既存コードを読んだ上で、ユーザー承認可能な計画へ落とし込む。</p>
<Steps>
  <Step number={1} title="EnterPlanMode">
    計画モードを開始して、読み取り専用ツールへ切り替える。
  </Step>
  <Step number={2} title="Explore">
    Glob / Grep / Read で既存コードと周辺コンテキストを徹底調査する。
  </Step>
  <Step number={3} title="Design">
    計画ファイルにステップバイステップで実行計画を記録する。
  </Step>
  <Step number={4} title="ExitPlanMode">
    計画レビューと承認を行い、実装フェーズへ移る。
  </Step>
</Steps>
