# Bonus: Remote Control

<p style={{ fontSize: "2rem", color: "var(--slide-text-muted)", marginTop: "-0.2rem" }}>スマホはセッションへの「窓」になるだけ。ネットワーク切断時も自動再接続される。</p>

```bash
/remote-control
```
