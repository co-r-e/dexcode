# Ralph Loop

<p style={{ fontSize: "2rem", color: "var(--slide-text-muted)", marginTop: "-0.2rem" }}>セッション終了をインターセプトし、タスク完了まで自動反復し続けるプラグイン。</p>

```bash
/ralph-loop "REST APIを構築"   --completion-promise "DONE"   --max-iterations 20
```
