# Get Started in 3 Steps

<Steps>
  <Step number={1} title="Install">
    Run `npm install -g dexcode` to install the CLI
  </Step>
  <Step number={2} title="Create a Deck">
    Generate a new deck from a template with `dexcode init my-deck`
  </Step>
  <Step number={3} title="Start Presenting">
    Live preview with `dexcode dev`, export to PDF with `dexcode export`
  </Step>
</Steps>
