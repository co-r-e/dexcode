# Design Presets

Pre-built theme configurations for quick selection. Present 3 options that match
the deck's mood. Users can pick one as-is or use it as a starting point.

## How to Present Presets

When suggesting presets, show a compact summary:

```
1. **Corporate Navy** - 信頼感・企業向け (白背景 + 紺)
2. **Tech Purple** - テック・AI向け (白背景 + 紫)
3. **Dark Mode** - インパクト重視 (黒背景 + 白)
```

Let the user pick by number or name. If they want to customize, use the picked
preset as a base and adjust specific colors.
