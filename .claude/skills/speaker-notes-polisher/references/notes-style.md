# Notes Style

`speaker-notes-polisher` が出力する `notes` の統一スタイル。

## 基本構成

```text
目的:
・このスライドで達成したいことを1文で示す

話すポイント:
・論点1
・論点2
・論点3

目安時間:
・45秒
```

## ルール

- セクション見出しは必ず `目的` / `話すポイント` / `目安時間` の順番。
- 箇条書きは `・` を使用。
- 1行は短く保ち、読み上げ時に息継ぎできる長さにする。
- `目安時間` は秒単位で記述（例: `30秒`, `45秒`, `60秒`）。
- 抽象語だけで終わらず、本文内の具体語（見出し・キーワード）を含める。
